Année de Terminale

  • Projet Python : Les objectifs de ce projet étaient de modéliser des cartes de jeu, gérer un paquet de cartes, simuler une partie de jeu entre deux joueurs et créer une interface utilisateur graphique en utilisant la programmation orientée objet (POO) en Python. Le projet comprenait la création de classes pour représenter les cartes (`Carte`), le paquet de cartes (`Paquet`) et les mains des joueurs (`Main`), ainsi que la simulation d'une partie de jeu avec des règles spécifiques. Une interface utilisateur graphique a été développée avec Tkinter pour permettre aux joueurs de saisir leurs noms, de démarrer la partie, d'afficher les cartes jouées, les scores en temps réel et le résultat final dans une boîte de dialogue pop-up.
