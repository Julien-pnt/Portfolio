# IMAGES À AJOUTER POUR LA PAGE PASSIONS

## Voitures de rêve (Section "Rêves Automobiles")
Placez ces images dans le dossier `PortFolio/Images/` :

1. **bmw-m3-cs.jpg** 
   - BMW M3 CS (couleur bleu ou gris)
   - Dimensions recommandées : 800x600px minimum
   - Format : JPG ou PNG

2. **porsche-911-turbo-s.jpg**
   - Porsche 911 Turbo S 
   - Dimensions recommandées : 800x600px minimum
   - Format : JPG ou PNG

3. **mercedes-amg-gt-r.jpg**
   - Mercedes AMG GT R (de préférence en vert ou noir)
   - Dimensions recommandées : 800x600px minimum
   - Format : JPG ou PNG

## Destinations de rêve (Section "Destinations de Rêve")

4. **miami-beach.jpg**
   - Vue de Miami Beach avec Ocean Drive ou architecture Art Déco
   - Dimensions recommandées : 1000x600px minimum
   - Format : JPG ou PNG

5. **nurburgring.jpg**
   - Circuit du Nürburgring ou une vue du célèbre "Enfer Vert"
   - Dimensions recommandées : 1000x600px minimum
   - Format : JPG ou PNG

## Notes importantes :
- Utilisez des images libres de droits ou vos propres photos
- Respectez les noms de fichiers exactement comme indiqués
- Les images seront automatiquement redimensionnées par le CSS
- Pour une meilleure qualité, privilégiez des images haute résolution

## Fallback :
Si une image n'est pas trouvée, le navigateur affichera le texte alternatif (alt).
Les anciens gradients colorés ont été conservés comme fallback en CSS.
