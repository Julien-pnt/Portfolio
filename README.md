## Installation et utilisation

### Prérequis
- Navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Serveur web local (XAMPP, WAMP) ou serveur Apache pour le déploiement

### Installation locale
1. Cloner ou télécharger le repository
2. Placer le dossier dans le répertoire du serveur web (htdocs pour XAMPP)
3. Accéder à `http://localhost/Portfolio/PortFolio/Html/Index.html`

### Déploiement en production
1. Transférer les fichiers vers le serveur web
2. Configurer les permissions des fichiers
3. Vérifier la configuration Apache (.htaccess)
4. Accéder au site via le nom de domaine configuré
